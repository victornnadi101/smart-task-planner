SMART TASK PLANNER APPLICATION

OVERVIEW:

The Smart Task Planner is a web-based application designed to help users manage tasks efficiently. It allows users to create, track, complete, and organize tasks with priority levels and deadline-based notifications.
This project was developed using Java Spring Boot for the backend and HTML/CSS/JavaScript for the frontend.

TECHNOLOGIES USED:

1. Java (Spring Boot)

2. HTML, CSS, JavaScript

3. SQLite Database

4. Maven

5. Render (Deployment)

FEATURES:

1. User registration and login

2. Add, view, complete, and delete tasks

3. Clear all tasks

4. Task priority system (High, Medium, Low)

5. Task status tracking (Pending, Completed)

6. Search and filter tasks

7. Notification system for upcoming deadlines

8. Responsive design (desktop and mobile)

HOW TO RUN THE APPLICATION:

Option 1: Run Online

Use the deployed version:
https://smart-task-planner-7csu.onrender.com

Option 2: Run Locally

PREREQUISITES:

1. Java JDK 17
2. Maven installed

STEPS:

1. Clone or download the project
2. Open the project in IntelliJ IDEA
3. Locate the main class: SmartTaskPlannerWebApplication.java
4. Run the application
5. Open browser and go to: http://localhost:8080

PROJECT STRUCTURE:

1. Controller Layer → Handles HTTP requests
2. Service Layer → Business logic
3. DAO Layer → Database operations
4. Frontend → HTML, CSS, JavaScript

KNOWN LIMITATIONS:

1. No session-based authentication
2. SQLite used instead of PostgreSQL
3. Notifications are in-app only
